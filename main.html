Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are multiple methods for choosing an element in the Document Object Model (DOM). Here are the most common methods:

1.getElementById():
An element is chosen using this method based on its id attribute. One element is returned. Example
const element = document.getElementById('elementId');

2.getElementsByClassName():
This method selects all elements that have a specific class name. It returns a live HTMLCollection of elements.Example:
const elements = document.getElementsByClassName('className');

3.getElementsByTagName():
This method selects all elements with a specified tag name. It returns a live HTMLCollection.Example:
const elements = document.getElementsByTagName('div');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent :-

innerHTML:

Gets/sets HTML content (including tags). It can be slow and exposes you to XSS risks.

innerText:

Gets/sets visible text, excluding HTML tags. Considers styling (e.g., hidden text isn’t retrieved). Slower than textContent.

textContent:

Gets/sets raw text, excluding HTML tags. Doesn’t consider styling (e.g., hidden text is included). Fastest.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element instead of individual child elements. The parent listens for events on its children using event bubbling, improving performance and handling dynamic elements easily.
Example:

document.getElementById('parent').addEventListener('click', function(event) {
if (event.target && event.target.matches('button')) {
console.log('Button clicked');
} });

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and "bubbles up" to its parent elements. This means the event is triggered on the target first, then propagates upward through the DOM tree, triggering event listeners on each ancestor element.
Example:

document.getElementById('parent').addEventListener('click', function() {
console.log('Parent clicked');
});

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements in JavaScript:

1.Create an element:

Example:
const newElement = document.createElement('div');


2.Add an element:

Example:
const parent = document.getElementById('parent');
parent.appendChild(newElement);


3.Remove an element:

Example:
newElement.remove();
